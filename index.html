About Task Timer

Task Timer is a project made for FreeCodeCamp. Based on the popular Pomodoro method, you can set times for how long you want to focus on a task on the right, and how long a break you'd like on the left. You can also make a list of the tasks you'd like to focus on below the timer and choose which task to focus on currently by selecting it. A sound will ring to remind you when the amount of time you've specified has elapsed.
